# Deploying OpenShift Container Platform 4.4 on VMware vSphere

OpenShift Container Platform is fully supported on VMware vSphere infrastructure. For installation of OpenShift Container Platform 4, VMware vSphere version 6.5 or 6.7U2 or higher version is required.

For more information on the detailed process of installation and infrastructure requirements, see https://docs.openshift.com/container-platform/4.5/installing/installing_vsphere/installing-vsphere.html
