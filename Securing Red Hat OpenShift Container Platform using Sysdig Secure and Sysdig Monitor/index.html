# Script Usage:

The python script filesplit.py is used to convert a word document into multiple md files split at headings

  • place the filesplit.py in the directory you want to create the required split md files
  • rename the required deployment guide word document as DeploymentGuide.docx and place it in the same directory
  • open command prompt navigate to the script location and execute the following command
    python filesplit.py
  • a markdown file guide.md file will be created along with multiple folders
  • the guide.md file is the markdown file version of the entire word document
  • the multiple folders created will be named according to the level one headings in the guide.md file.
  • each of these folders will have their respective markdown files in it

NOTE After cloning the directory this README.md can be modified to whatever the user wants displayed on the Landing Page of the website.

# Hosting the Web Doc

Steps to host the Deployment Guide

  • Prerequisites
    npm install -g vuepress # OR yarn global add vuepress
  • Navigate into the docs folder
  • Start development server
    vuepress dev
  • Create Build
    vuepress build
