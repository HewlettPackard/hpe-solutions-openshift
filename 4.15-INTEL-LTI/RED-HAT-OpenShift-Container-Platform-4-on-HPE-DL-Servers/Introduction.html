# INTRODUCTION

The Red Hat® OpenShift® Container Platform (RHOCP) is a dependable hybrid cloud foundation for developing and scaling containerized applications. It provides a unified, consistent Kubernetes platform anywhere the Red Hat® Enterprise Linux® (RHEL) operating system is installed. Container platforms built on Kubernetes deliver a cloud-like experience across all the locations where you deploy it, whether it is a private or public cloud, on-premises, or at the edge.

This document describes the deployment of a highly available and secure Red Hat® OpenShift® Container Platform 4.15 on HPE ProLiant DL360 and DL380 Gen11 servers. It is intended to be used with the Ansible® Playbook and python scripts found at https://github.com/HewlettPackard/hpe-solutions-openshift.git (opens new window). This document provides implementation guidelines for RHOCP 4.15. For more information, see the https://docs.openshift.com/container-platform/4.15/welcome/index.html (opens new window). Hewlett Packard Enterprise strives to make regular updates to this solution with the goal of validating it against the latest available version of RHOCP. As a result, the references in the document will generally refer to RHOCP 4.15 and mention specific sub-versions in strategic locations to note the tested version.

The RHOCP 4.15 deployment on HPE ProLiant DL360 and DL380 Gen11 servers includes the following:

  • RHEL 8.9 is installed on the HPE ProLiant DL360 Gen11 servers.
  • Three RHOCP master nodes run as virtual machines on Kernel-based Virtual Machine (KVM) nodes and three HPE ProLiant DL360 Gen11 servers are used for the worker nodes (out of which one node acts as both, the installer and the worker node).

RHOCP 4.15 deployment supports below storage options

The RHOCP 4.15 deployment on HPE ProLiant DL380 Gen11 servers includes the following:

  • RHEL 8.9 is installed on the HPE ProLiant DL380 Gen11 servers for Openshift Data Foundation(ODF). After os deployment for odf we will add node to existing cluster and deploy odf on nodes.

  • HPE Container Storage Interface (CSI) Driver with HPE Alletra storage is deployed on the existing RHOCP 4.15 solution offering.

It is recommended this document should be reviewed in its entirety and the installation user should understand all prerequisites and procedure before installation. It is also recommended that the installation user review the OpenShift Container Platform 4.15 installation process as described by Red Hat.
